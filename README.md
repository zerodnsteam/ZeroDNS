## ⚡️ ZeroDNS란?

- 여러 글로벌 DNS 필터 소스를 **매일 자동 병합·중복 제거·최적화**
- 최신 광고, 트래커, 스팸, 피싱 도메인 실시간 차단

## ⏬ **설치/적용 예시**

1. **AdGuard (iOS/Android/브라우저/PC)**  
   - `사용자 DNS 필터` → 위 Cl주소 복사/붙여넣기  
2. **AdGuard Home / Pi-hole / 라우터 / 기타**  
   - “커스텀 블록리스트” 입력란에 위 주소 추가  
3. **자동 업데이트**  
   - 매일/매주 자동 최신 필터 적용
